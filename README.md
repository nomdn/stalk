## 特点
受监视服务器支持正反向  
正向：  
在该计算机上运行web服务器，由主api服务器访问
反向：
向主api服务器发送信息  
有一半fastapi
## 把这个项目部署在你朋友家，再视奸他
**部署方法:**  

1. 下载本仓库
2. 根据需要安装依赖
3. 运行你需要的服务器
## 注意事项
推荐python版本:python 3.12  
可用版本> 3.10,<4  
PC API需要安装pywin32
